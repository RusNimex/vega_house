# Vega House API

REST API для мобильного приложения - осмотр объектов недвижимости и хранение данных по ним.

## Технологии

- **Laravel 11** - PHP фреймворк
- **MySQL 8** - База данных
- **Laravel Sanctum** - Аутентификация для API
- **OpenAPI** - Документация по эндпоинтам

## Использование API

Полная документация API доступна в формате OpenAPI (Swagger): [swagger.yaml](swagger.yaml)

> **Примечание:** Информация о развертывании, установке и командах Sail находится в файле [DEPLOY_local.md](DEPLOY_local.md).

## Лицензия

Приложение является тестовым. Развернул для собственного развития, пощупать новую версию лары в связке с sanctum. Пробуем создать API по мотивам одного макета в свободном доступе. Добавлю тесты и их запуск.
